# 🌿 Wellness Tracker

A personal calorie tracker built for a 44-year-old woman on a wellness journey.

- AI-powered meal logging — just describe what you ate
- 1,400 cal/day goal
- Daily motivational quotes
- Edit/clear meals anytime

## Setup

```bash
npm install
npm run dev
```

## Deploy
Push to GitHub, then connect to [Vercel](https://vercel.com) for free hosting.
